ℹ️ Informations

Bienvenue sur le gestionnaire de pêche !

Fonctionnalités :

  • ✅ Cochez les poissons capturés
  • 🔍 Filtrez par état, lieu, moment et météo
  • 📊 Suivez votre progression en temps réel
  • 🗂️ Triez les colonnes en cliquant sur les en-têtes

Vos données sont sauvegardées localement dans votre navigateur.

🔄 Réinitialiser mes données

Pour réinitialiser uniquement les poissons pêchés :

  1. Appuyez sur F12
  2. Allez dans l'onglet Console
  3. Entrez la commande localStorage.removeItem('caughtFish')
  4. Rechargez la page

Pour réinitialiser uniquement les notes personnelles :

  1. Appuyez sur F12
  2. Allez dans l'onglet Console
  3. Entrez la commande localStorage.removeItem('siteContent')
  4. Rechargez la page

Pour réinitialiser toutes vos données :

  1. Appuyez sur F12
  2. Allez dans l'onglet Console
  3. Entrez les commandes localStorage.removeItem('caughtFish') & localStorage.removeItem('siteContent');
  4. Rechargez la page

🔐 Sécurité des données

Vos données sont stockées uniquement dans ce navigateur. Un changement d'appareil, de navigateur, ou un vidage du cache les effacera. Exportez-les régulièrement pour les garder en sécurité.

📤 Exporter mes données

Télécharge un fichier .json contenant tes poissons capturés et tes notes personnelles.

📥 Importer mes données

Restaure des données précédemment exportées (remplace les données actuelles).
